# Logo Reference

## Logo Assets

The EDUCORE AI logo should be used as provided. The logo consists of:

1. **Circular Logo Graphic**: 
   - Dynamic, swirling green graphic with interconnected leaf-like/ribbon-like elements
   - Colors: Light lime green to deep forest green gradient
   - Central white/pale blue circle with dark green letter "E"

2. **Text**: "EDUCORE AI" in uppercase, sans-serif, dark green

## Logo Placement

Based on design system:
- **Splash/Landing**: 240px width × 64px height, centered
- **Header**: 160px width × 40px height, left position
- **Compact**: Can use 1:1 aspect ratio for square format

## Important

⚠️ **DO NOT MODIFY THE LOGO**
- Keep logo exactly as provided
- Use original logo files when available
- Do not apply filters or transformations unless specified

## Logo Files Location

Logo files should be stored in:
- `frontend/public/logo/` (for static assets)
- Or referenced from design system assets
